# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

[しゅんけー](https://x.com/shunk031) が Claude Code と一緒にブログ記事を執筆するためのリポジトリ。

## ディレクトリ構成

- `articles/` - Zenn に公開する記事のマークダウンファイル（.md）を格納
- `images/` - 記事で使用する画像ファイルを格納（記事ごとにサブディレクトリを作成）

## ブログ執筆ルール

AI っぽさを排するため以下を守ること：

- 太字（`**text**`）は一切使用しない
- 見出しに絵文字やコロンは一切使用しない
- 箇条書きは最小限に。説明文は短い体言止め文ではなく、通常の語り文章のベタ書きにする
- 表も最低限の利用に止める
- 見出し（`#`）やコードブロックは普通に使って OK
- リンク埋め込み、画像挿入も普通に使う
- 執筆前に必ず「`@articles/`」内の記事を読み、しゅんけーの文体を参考にする

## 文体ルール

- 見出しにコロン（`:`）は使わない
- 見出し内の番号付き項目には墨付き括弧【】を使い、その後に半角スペースを入れる
  - 例: `## 【ハマりポイント1】 プロンプトキャッシュが使えない`
- 文中のバッククォート（ `code` ）の前後には半角スペースを入れる
  - 例: "通常の `data` イベントに加えて"
- バッククォート以外のアルファベット（通常の英単語）の前後には半角スペースを入れない
  - ○: `Claudeと比較すると` / ×: `Claude と比較すると`
